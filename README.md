## Краткая инструкция по запуску и разворачиванию проекта

### Установка зависимостей

```sh
npm install
```

### Запуск в режиме разработки

```sh
npm run dev
```

### Собрать проект перед разворачивание на прод

```sh
npm run build
```

### Запуск собранного приложения на localhost

```sh
npm run preview
```
